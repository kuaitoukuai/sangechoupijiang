《三个臭皮匠》一个同时向多个AI提问的python程序

1、双击程序，会启动Edge、Chrome浏览器

2、将要浏览的AI页面手动打开、收藏夹批量打开，登陆自己的帐号

3、可以点击“检查已打开标签”查看有多个标签页是否正确

4、输入提问内容，点击提交所有AI或者当前AI
